# Skills for Data-Intensive Research — GitHub Organisation

This GitHub organisation supports the delivery of the University of Melbourne subject:

**BINF90018 — Skills for Data-Intensive Research**  
Master of Bioinformatics  
School of BioSciences / Melbourne Bioinformatics

The organisation hosts repositories used for **teaching, tutorials, and assignments** associated with the subject.

---

## About the subject

Skills for Data-Intensive Research is a **4-week intensive subject** focused on the practical infrastructure required for modern computational research.

The subject introduces students to practices that enable reproducible and collaborative research, including:

- command-line environments
- remote computing
- version control with Git
- research data management
- reproducible project structure
- scientific software practices
- workflow orchestration

The emphasis is on **doing simple things correctly**, rather than advanced data analysis or large-scale software engineering.

## What you will find in this organisation

This organisation hosts repositories used during the subject, such as:

- workflow and infrastructure examples
